Datacenters¶

We operate several Datacenters around the world. Each datacenter holds all information for one or more regions of the game. Currently all data is located in DC01, but this may change in the future. Users will be notified prior to the change and an API endpoint will be made available to provide metadata describing where each region’s data can be found.
